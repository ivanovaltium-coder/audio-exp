Классификация выполняется с помощью ансамблевой модели (Random Forest), обученной на мел-кепстральных коэффициентах (MFCC) и их динамических производных.

## Возможности
- **Высокая точность:** Использование частоты дискретизации 96 кГц позволяет улавливать высокочастотные гармоники, невидимые для стандартных микрофонов.
- **Низкая задержка:** Прямой доступ к оборудованию через ASIO исключает задержки ОС.
- **Масштабируемость:** Поддержка до 8 синхронизированных каналов для будущего пеленгования источника звука.
